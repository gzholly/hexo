目送

年轻求得圆满／随着岁月走散 / 忍不住回头看／剩下的只是片段／
生命不断转弯／起起落落变成习惯／爱情像是考验／从不承诺永远

老婆与女儿去了健身，家突然静了下来。

我开了电脑，打开MediaPlayer，转出一首蔡琴<缺口>，然后翻起书桌上的书看起来。

我看了几篇，突然发现，很小机会在晚上、在这个时候，在看纸质的传统书籍——既然是突然有悟，当然要写下为记。

书是龙应台的《目送》。

我有次将其中一篇《目送》读给女儿听，女儿话：“哇，都写得几好”。

“哈哈，几好？你写得出吗？不如试下你写啦”我说。

“好啊，下次啦。”

看着女儿，我想起她第一次背起一个维尼熊书包——书包里面尽是要换的衣裤毛巾，小小的手，紧紧圈着我的颈，满眼尽是泪光，抽泣不停。那年，她是幼儿园小班的学生。

老师接过女儿，我无比清楚地看着自已的孩子背影——纵千万人，你会准确分辩得出，她瘦小的背影消失在门后，我仿佛仍听到她轻轻的哭声。

当女儿再次背起一个公主书包，道声“ByeBye”，匆匆地上学了，我在她身后说“小心D”，然后目送她下楼。

我慢慢了解到：老奶奶以前最喜欢说“好冻啊！着多件衫啊！”无论你穿多少衣服，她都说好冻，要你穿多件。就如今日我让女儿穿多件衣服一样。

我也慢慢了解到：
老妈看着我背影，我看着女儿背影，一代一代，一场父女母子，不过是今生今世不断地目送他的背影渐行渐远。

这就是——目送。

这些年像陀螺一样旋转 / 爱恨都变得无关
过去的风雨留给别人评断 / 无愧了一切都平淡
是有一点遗憾 / 幸福没有答案 / 付出不能计算 / 谁能够抚平背叛
不必再去感叹 /　要笑着把眼泪擦干 / 夜晚是个难关 / 寂寞需要勇敢
